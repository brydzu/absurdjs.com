Superpowers


AbsurdJS is a JavaScript library available for Node.js and browser usage. It acts as a CSS and HTML preprocessor. There are a lot of build-in mixins called atoms and molecules. Having both, CSS and HTML preprocessing, you are able to create cutting edge web components.
All for JavaScript and JavaScript for all.

Contribute

Run the client-side port tests and report if they fail.
